I'd like to share my first dashboard created in Power BI.

This project focuses on manufacturing process analysis and is divided into three main sections:

🔹 Production Performance – presents key production KPIs, including the total number of jobs, completed and delayed jobs, as well as the Completion Rate. It also provides a breakdown of production results by shift.

🔹 Production Details – offers a more detailed analysis of the production process, including machine performance, operation types, and average processing time.

🔹 Delay Rate Analysis – highlights the delay rate for each machine and production shift, making it easier to identify areas that require improvement.
